Notes

  • The counter is global — every saved quote increments it by one, regardless of user.
  • A quote number is assigned once on first save and never changes — re-saves keep the original.
  • Changing the template only affects quotes numbered after the change.
  • Use Python format specs for padding, e.g. {{ '{counter:04d}' }} → 2547 becomes 2547, 7 becomes 0007.
  • Default template is {{ '{user_initial}{counter}/{month}/{year}' }} producing B2547/04/2026.
